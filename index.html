Infinix Hot 50 Pro Plus (X6880)

Welcome to The Ground Zero

Ini adalah repositori pusat untuk daftar *Custom ROM* perangkat X6880. Proyek ini dikelola oleh ASNF Techno Lab. Seluruh data, instruksi *flash*, *bug list*, dan kredit *developer/porter* tersimpan dengan aman dalam ekosistem Shinko Engine 15.

⚠️ Peringatan (Disclaimer)

*Flashing Custom ROM* membawa risiko *bootloop* atau kerusakan data. Pastikan Anda sudah mengerti cara menggunakan *Custom Recovery* dan sudah melakukan *Backup Data*. Segala risiko ditanggung oleh pengguna.

Standard Operating Procedure

Metode Standar (.zip)

Gunakan metode ini untuk ROM berbasis .zip (HyperOS 1, 2, HelloUI, dll).

  1. Reboot ke Custom Recovery.
  2. Flash file .zip ROM yang sudah di-download.
  3. Lakukan Format Data (Ketik "yes").
  4. Reboot System.

Metode Super.img (HIOS/XOS/RealmeUI)

Untuk ROM port dari Transsion / BBK yang berbentuk ekstrak Super.img.

  1. Reboot ke Custom Recovery.
  2. Flash Kernel Permissiver.
  3. Flash Super.img (yang sudah diekstrak dari file .zip) ke partisi Super.
  4. Lakukan Format Data.
  5. Reboot System.

*Jika Anda update dari versi sebelumnya (misal XOS 16.1 ke 16.2), Format Data opsional, cukup Flash Permissiver & Super.img.

Informasi GKI Permissiver

Beberapa ROM mewajibkan status SELinux Permissive untuk bisa booting. Pastikan Anda men-*download* dan men-*flash* file Permissiver GKI yang disarankan oleh Porter sebelum *reboot*.
